TuskVector

TuskVector is an API framework for vector operations, emphasizing Python-based backend and HTML/HTMX for frontend interactions.

Key Features:

  • Vector embedding and storage
  • Similarity search using HNSW index
  • Integration with ada02 text embedding and GPT 4o
  • API key management

Tech Stack

  • Python
  • FastAPI for API development
  • SQLAlchemy with PostgreSQL and pgvector extension
  • OpenAI API for embeddings and LLM queries
  • HTMX for frontend interactivity (JavaScript-free approach)

The goal of this project was to a robust application with minimal boilerplate and without relying on JavaScript, focusing instead on Python backend efficiency and HTMX for frontend interactions.
