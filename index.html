TuskVector - API Platform for Vector Operations

Python pgvector HNSW ada2 FastAPI HTMX

TuskVector is an API framework that transforms data into 1536D vectors, employs HNSW indexing for efficient information retrieval, and enhances databases with advanced search capabilities.

Key Features and Tech Stack:

  • Backend: Python with FastAPI
  • Vector Operations: OpenAI's text-embedding-ada-002 model for 1536D embeddings
  • Search: HNSW index with cosine similarity (configurable M=16, ef_construction=64)
  • Database: PostgreSQL with pgvector extension, using SQLAlchemy ORM
  • LLM Integration: OpenAI API for embeddings (ada2) and LLM queries (GPT 4o)
  • Frontend: HTMX for JavaScript-free interactivity
